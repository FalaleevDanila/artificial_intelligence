\doxysubsection{Detailed Description}
Класс игрового поля, в нем будет происходить сама игра 

Данный класс отвечает за создание всех объектов (таких как \mbox{\hyperlink{class_player}{Player}}, \mbox{\hyperlink{class_npc}{Npc}} и Walls), за их взаимодействие, является посредником передачи информации между объектами типа \mbox{\hyperlink{class_man}{Man}}
